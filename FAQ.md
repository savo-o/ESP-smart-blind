# FAQ - ESP SmartBlind

## ❓ После прошивки пропали настройки Wi-Fi и ключ
Это нормально.

Полная прошивка (FULL) очищает память настроек.
Используйте OTA-прошивку для обновлений - она сохраняет настройки.

## ❓ Чем отличается FULL и OTA прошивка?

FULL прошивка
- используется при первой установке
- прошивается через USB или онлайн прошивку
- сбрасывает настройки

OTA прошивка
- используется для обновлений
- устанавливается через веб-интерфейс ESP
- сохраняет настройки

## ❓ Можно ли прошить OTA файл через онлайн прошивку?
Нет.

OTA файл предназначен только для обновления через уже установленную прошивку.

Если прошить OTA как FULL - устройство может не загрузиться.

## ❓ Онлайн прошивка не работает
Проверьте:

- используется Chrome или Edge
- сайт открыт по HTTPS
- плата подключена по USB
- выбран правильный COM-порт

Онлайн прошивка не работает в Firefox и на телефонах.

## ❓ Устройство не подключается к Wi-Fi
Включите режим настройки:

- подключитесь к точке ESP-SmartBlind-Setup
- откройте браузер
- введите данные Wi-Fi
- сохраните настройки

## ❓ Где посмотреть IP-адрес ESP?
IP-адрес отображается:

- в логах Serial Monitor
- на главной странице веб-интерфейса
- по адресу http://smartblind.local

## ❓ Я забыл ключ!
Зажмите кнопку BOOT на 6 секунд, до рывка сервопривода. Это сбросит настройки. 
Повторите процедуру первой настройки.

## ❓ Что-то не работает корректно.
Зажмите кнопку BOOT на 6 секунд, до рывка сервопривода. Это сбросит настройки. 
Повторите процедуру первой настройки. Скорее всего, проблема исправится. Если нет - оставьте проблему в Issue GitHub-а

## ❓ Можно ли использовать без Яндекс Алисы?
Да.

Устройство полностью работает через HTTP-запросы.

Алиса - это дополнительная возможность.

## ❓ Поддерживается ли Zigbee?
Нет.

Проект основан на Wi-Fi (ESP32).

Возможно добавление Zigbee в будущих версиях.

## ❓ Можно ли использовать проект в коммерческих целях?
Нет.

Проект распространяется только для личного использования.
Коммерческое использование запрещено лицензией.
