## 🪟 ESPSmartBlind

DIY-проект умной римской шторы на базе ESP32  
с управлением через HTTP и Яндекс Алису.
Проект находится в доработке.

## ⚠️ Важно

Проект остановлен на неопределнный срок.

## ⚠️ Важно

Проект ориентирован **на программную часть устройства**.

Механическая реализация (двигатель, колесо, прижим, корпус, тип цепочки)  
зависит от конкретных штор и условий установки и **не входит в репозиторий**.

Автор тестировал проект на экспериментальном стенде.  
Для стабильной работы рекомендуется использовать:

- мощные сервоприводы  
- готовые моторизованные механизмы  
- или собственные доработанные крепления  

Проект открыт для доработок и экспериментов.

## 🪟 Какие устройства поддерживаются

## Несмотря на название, проект не ограничивается только шторами.

По своей сути это универсальный контроллер для механизмов с двумя состояниями (открыто / закрыто), который может быть использован для различных задач автоматизации.

Проект может применяться для:
- рулонных штор
- римских штор
- любых штор с пластиковой шариковой цепью
- других механизмов, которые можно открывать и закрывать (люки, заслонки, экраны и т.п.)

Ключевое требование - возможность реализовать механическую часть (например, с помощью 3D-печатного колеса или другого привода).

## 🔧 Используемые элементы

- ESP32
- сервопривод с непрерывным вращением (360°, continuous rotation)
- провода Dupont
- механический привод (например, 3D-печатное колесо под шариковую цепочку)

## ⚙️ Необязательные элементы

- концевой выключатель (на данный момент не используется)
- понижающий DC-DC преобразователь (при необходимости)
- макетная плата (по желанию)

## 🧠 Принцип работы

ESP32 управляет сервоприводом, который вращает колесо,
зацепленное за шариковую цепочку римской шторы.

Устройство принимает команды через HTTP или голосового ассистента,
после чего выполняет соответствующее действие.

Концевой выключатель используется для защиты механизма
от превышения крайних положений.
